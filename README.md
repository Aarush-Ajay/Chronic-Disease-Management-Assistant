# Chronic-Disease-Management-Assistant
This is a multi-agent AI for chronic care using IBM Granite and RAG. It scrubs patient logs, detects 15% vital escalations via a Trend Detective, and provides clinical guidance using Chroma DB. It turns "dormant" data into early alerts for diabetes and hypertension.
